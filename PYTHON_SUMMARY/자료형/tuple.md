자료형
========

튜플
----------
#### 리스트는 값의 생성, 삭제, 수정이 가능하지만, **튜플은 값을 바꿀 수 없음**
```python
a = ()
b = (1, 2, 3)
c = ('a', 'b', ('c', 'd'))
d = 1, 2, 3 # 괄호 생략 가능
e = (1, ) # 한 개의 요소만 있을 경우, 콤마(,)를 붙여야 함
```

* 튜플 다루기
  + 인덱싱 (Indexing)
    ```python
    a = (1, 2, 'c', 'd')
    a[0] # 1
    a[3] # d
    ```

  + 슬라이싱 (Slicing)
    ```python
    a = (1, 2, 'c', 'd')
    a[1:] # (2, 'c', 'd')
    ```

  + 튜플 더하기
    ```python
    a = (1, 2, 'c', 'd')
    b = (3, 4)
    a + b # (1, 2, 'c', 'd', 3, 4)
    ```

  + 튜플 곱하기
    ```python
    a = (1, 2)
    a * 3 # (1, 2, 1, 2, 1, 2)
    ```

  + 튜플 길이 구하기
    ```python
    a = (1, 2, 'c', 'd')
    len(a) # 4
    ```
