자료형
========

자료형의 값을 저장하는 공간과 변수
----------
* 변수 생성 방법
  ```python
  변수 이름 = 변수에 저장할 값
  ```

* 변수란?
  #### 객체를 가리키는 것
  ```python
  # [1, 2, 3] 값을 가지는 리스트 자료형(객체)이 자동으로 메모리에 생성
  # 변수 a는 값이 저장된 메모리의 주소 가리킴
  a = [1, 2, 3]
  id(a) # a 변수가 가리키는 메모리 주소
  ```

* 리스트 복사
  ```python
  a = [1, 2, 3]
  b = a # 해당 리스트를 참조하는 변수가 늘어난 것
  # id(a)와 id(b)는 동일
  ```
  ```python
  # is 명령어 : 동일한 객체를 가리키는지 판단
  a is b # True
  ```
  ```python
  # 동일한 객체를 가리키므로 둘 다 바뀜
  a[1] = 4
  a # [1, 4, 3]
  b # [1, 4, 3]
  ```
  + [:] 이용
    ```python
    a = [1, 2]
    b = a[:] # 복사
    a[1] = 3
    a # [1, 3]
    b # [1, 2]
    ```

  + copy 모듈 이용
    #### > from copy import copy 사용
    ```python
    a = [1, 2]
    b = copy(a) # b = a[:]와 동일
    ```

* 변수를 만드는 여러 가지 방법
  + 튜플
    ```python
    a, b = ('python', 'C++')
    (a, b) = 'python', 'C++'
    ```

  + 리스트
    ```python
    [a, b] = ['python', 'C++']
    ```

  + 여러 개 변수를 같은 값에 대입
    ```python
    a = b = 'python'
    ```

  + 변수 값을 간단하게 바꾸는 방법
    ```python
    a = 1
    b = 2
    a, b = b, a
    a # 2
    b # 1
    ```
